Detailed Description

This parameter struct generating field boudary values. The values are superpositions of Gaussian pulses of the form: peakamps cos( omegas * t + phases) exp(-(t-delays)^2*invWidths^2) The coefficients peakamps, omegas, phases, delays,and invWidths are double arrays of length nwaves.
